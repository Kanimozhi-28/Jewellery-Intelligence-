# Jewellery Shop Interaction Intelligence System

Comprehensive project for tracking customer interactions and sales intelligence in jewellery retail.

## Project Structure

This repository consolidates several components of the system:

- **admin-portal**: The management dashboard for administrators to monitor store performance, manage salespersons, and view customer analytics.
- **admin-backend**: Backend services for the administration system.
- **mobile-app**: The mobile application for salespersons to record customer interactions.
- **mobile-backend**: Backend services supporting the mobile application.
- **salesperson-portal**: A web-based portal for salespersons.
- **unified-app**: A unified application interface.
- **docs/**: Project documentation including PRDs, Technical Architecture, and Guides.
- **database_setup.sql**: SQL script for initializing the system's database.

## System Components

### 1. Admin Portal
- **Dashboard**: Real-time sales and traffic KPIs.
- **Management**: User and salesperson management.
- **AI Intelligence**: Analysis of jewellery performance and customer comments.
- **Lost Opportunities**: Identification of items with high interest but low conversion.

### 2. Mobile Application
- Guided interaction tracking for sales staff.
- Session management for customer visits.
